## Resumen rápido

Las mejores zapatillas de crossfit según perfil:

- **Mejor general** → Nike Metcon 9 (~€140)
- **Más versátil** → Reebok Nano X4 (~€145)
- **Para principiantes** → Reebok Nano Court (~€95)
- **Levantamiento serio** → NoBull Trainer+ (~€140)

El crossfit es un deporte híbrido — combina levantamiento, gimnasia, cardio y carrera corta. La zapatilla correcta debe equilibrar estabilidad para pesos, flexibilidad para gimnasia y amortiguación para impacto. Una zapatilla de running NO sirve para crossfit.

## Qué hace distinta a una zapatilla de crossfit

- **Plataforma plana y estable** — drop bajo (0-4mm) para levantamientos
- **Talón rígido** — para box jumps y squats con peso
- **Suela durable** — resiste cuerda escalonada y burpees
- **Flexibilidad en antepié** — para movimientos dinámicos
- **Soporte lateral reforzado** — para movimientos multidireccionales

## Por qué NO usar zapatillas de running

**Errores graves al usar running para crossfit:**

1. **Talón blando** → inestabilidad en squats y deadlifts (riesgo de lesión)
2. **Drop alto (10-12mm)** → genera ángulo incorrecto en levantamientos
3. **Plataforma estrecha** → poco soporte lateral para movimientos multidireccionales
4. **Suela suave** → se destruye con burpees y cuerda escalonada

## Preguntas frecuentes

**¿Una zapatilla de crossfit sirve para correr?**
Para 1-3 km dentro de un WOD sí. Para entrenamientos de running puro (5K+) no — la falta de amortiguación generará molestias.

**¿Cuánto duran las zapatillas de crossfit?**
6-12 meses con uso regular (3-4 veces por semana). Cuerda escalonada y burpees son los enemigos.

**¿Vale la pena el precio de NoBull?**
Solo si hacés crossfit muy serio. Para crossfitters recreacionales, Reebok Nano X4 ofrece similar valor a menor precio.

**¿Mismas zapatillas para gym tradicional y crossfit?**
Sí, pero las de crossfit están mejor diseñadas para movimientos compuestos. Las de gym puro a veces sacrifican estabilidad por comodidad.
